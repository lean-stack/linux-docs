Linux hat eine lange Tradition der Selbsthilfe. Die Abkürzung **RTFM** (*Read The Friendly Manual*) ist nicht zynisch gemeint – sie spiegelt eine echte Stärke des Systems wider: Nahezu jedes Werkzeug bringt seine vollständige Dokumentation mit.

Wer weiß, wo er nachschlagen muss, kommt ohne fremde Hilfe weit. Dieser Abschnitt zeigt, welche Quellen es gibt und wann man welche nutzt.

## Überblick der Quellen

| Quelle | Wo | Am besten für |
|---|---|---|
| `--help` | Shell | Schneller Blick auf Optionen |
| `man` | Shell | Vollständige Referenz eines Befehls |
| `info` | Shell | GNU-Werkzeuge (zunehmend veraltet) |
| [TLDP](https://tldp.org) | Web | Konzepte, Guides, HOWTOs |
| Suchmaschine | Web | Fehlermeldungen, konkrete Probleme |
| KI | Web | Erklärungen, Skizzen, Einstieg |

- [Hilfe in der Shell](shell) – `--help`, `man`, `info`, `whatis`, `apropos`
- [Hilfe im Web & mit KI](web) – TLDP, Suchmaschinen, KI-Tools
